# app/mastery/

Per-KC **Bayesian Knowledge Tracing** plus the anti-gaming augmentation rules that
gate a mastery declaration: BKT threshold, minimum scored attempts, representation
diversity, an unscaffolded correct, interleaving, and the engagement floor.

Deterministic and pure — **no LLM, no DB, no network**. The five personas are its
integration tests; if a persona who shouldn't reach mastery does, this is broken.
