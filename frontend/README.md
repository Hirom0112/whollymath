# frontend/

React + TypeScript + Vite app: the learner-facing adaptive UI, the custom SVG math workspace (FractionBar, NumberLine, SymbolicEditor), the surface state machine (S1–S5), and the API client. A pnpm workspace package.

Does NOT live here: business logic, math verification, or mastery rules (those are backend, `backend/`); generated API types (those are `shared-types/`); infrastructure (`infrastructure/`).
